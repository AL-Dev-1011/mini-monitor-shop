# Mini Monitor Shop

Mini ecommerce web application built with Laravel + Tailwind CSS.

## Project Overview

This project is a mini ecommerce system focused on monitor products.

---

# Features

## Guest

- Browse products
- Product detail page
- Product filtering
- Product sorting
- Add to cart
- Buy now

---

## User

- Authentication
- User profile
- Checkout system
- Order history
- Order detail

---

## Admin

- Admin dashboard
- Product CRUD
- Order management
- Update order status

---

# Product Filters

- Brand
- Application
- Price Range
- Display Size
- Resolution
- Refresh Rate
- Panel Type
- Connection Types
- Aspect Ratio
- Response Time

---

# Tech Stack

- Laravel 12
- PHP 8.2
- Tailwind CSS
- MySQL
- Blade Components
- XAMPP
